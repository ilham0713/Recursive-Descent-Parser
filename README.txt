To compile & run this code, download the parsercodegen.c file and any requisite 
input files to the same folder/directory.
Use "gcc -g parsercodegen.c" to generate a program file from the .c file.

Run the program while passing in your input file with this 
command: "./a.out input_file.txt".

The expected output will be shown directly in the terminal.
